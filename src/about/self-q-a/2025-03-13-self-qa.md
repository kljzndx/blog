# 批判日记

## 批判列表

1. 谁让你感到愤怒、挫折、迷惑？为什么？谁激怒了你？他有哪些地方是你不喜欢的？
    - 我很烦躁，因为我爸总是在刷短视频
2. 你要他如何改变，你期待他怎么表现？
    - 我要他多出门转转，多干点别的事
3. 他应该（或不应该）做、想、成为或感觉什么呢？你想给他什么样的忠告？
    - 他不应该把声音开那么大
4. 你需要他怎么做，你才会快乐？
    - 我需要他少玩手机
5. 此刻，他在你心目中是怎样的人呢？请详细描述一下。
    - 一个只顾自己享受不管别人感受的人
6. 你再也不想跟这个人经历到什么事？
    - 我再也不想看到他躺在床上刷手机

## 对质列表

### 他总是在刷手机

1. 那是真的吗？
    - 是的
2. 你能百分之百肯定那是真的吗？
    - 不能
3. 当你持有那个想法时，你会如何反应呢？
    - 我会很烦躁，想把他手机砸烂
4. 没有那个想法时，你会是怎样的人呢？
    - 我的心情会很平静
5. 反向思考：
    - 我总是在打扰自己

### 我要他多出门转转

1. 那是真的吗？
    - 是的，我经常这么想
2. 你能百分之百肯定那是真的吗？
    - 不能
3. 当你持有那个想法时，你会如何反应呢？
    - 我感觉烦不胜烦，想着怎么把他打发走，好让我休息一会儿
4. 没有那个想法时，你会是怎样的人呢？
    - 我会很平静的接着休息
5. 反向思考：
    - 我总是要我自己保持警觉
    - 我总是在驱赶我自己
